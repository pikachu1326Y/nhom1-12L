Luyện tập 2 trang 105 Tin học 12: Thay đổi định dạng và màu sắc của phông chữ trong các vùng khi di chuyển chuột qua.

Để thay đổi định dạng và màu sắc của phông chữ trong các vùng khi di chuyển chuột qua, em có thể sử dụng hiệu ứng hover trong CSS.

Hover Effect
Di chuột qua đây để thay đổi
/* Định dạng mặc định */ .hover-area { font-size: 20px; color: black; font-family: Arial, sans-serif; padding: 20px; border: 1px solid #ccc; text-align: center; transition: all 0.3s ease-in-out; } /* Khi di chuột qua */ .hover-area:hover { font-size: 24px; /* Tăng kích thước chữ */ color: white; /* Đổi màu chữ */ background-color: #007BFF; /* Đổi màu nền */ font-weight: bold; /* In đậm chữ */ font-style: italic; /* Chữ nghiêng */ border-color: #0056b3; /* Thay đổi màu viền */ cursor: pointer; /* Biểu tượng con trỏ */ }
